MGU：最一般合一算法
合一(unify)是找到一组赋值，使得两个原子公式等价的过程。最一般合一就是找到能使得两个公式等价的最简单的赋值。
以P(x,b)和P(a,y)为例，最一般合一是{x = a,y = b}

实现MGU算法的步骤如下：
1. 如果两个公式都是变量，那么返回{x = y}。
2. 如果一个公式是变量，另一个公式不是变量，那么返回{x = t}
3. 如果两个公式都是常量，并且相同，那么返回空集。
4. 如果两个公式都是常量，并且不同，那么返回失败。
6. 如果两个公式都是复合公式，并且它们的谓词符号不同，那么返回失败。
7. 如果两个公式都是复合公式，并且它们的谓词符号相同，但是参数的数量不同，那么返回失败。
8. 如果两个公式都是复合公式，并且它们的谓词符号相同，并且参数的数量相同，那么对它们的参数进行递归调用MGU算法，并将结果合并。

需要实现的功能：
1. 根据输入的`str`判断谓词、函数、常量、变量
2. 获取公式的谓词符号和参数
3. 实现MGU算法的递归调用和结果合并
4. 将结果作为`dict`返回，表示变量的赋值关系
